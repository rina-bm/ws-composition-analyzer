# Web Service Composition Analyzer

This project is dedicated to modeling and analyzing compositions of web services.
It was developed as part of a bachelor's thesis focused on evaluating the
characteristics of composite service systems.

The system allows simulation of web service workflows and calculation of key
metrics such as execution time, reliability, and performance.

---

## Project Goal

The goal of this project is to develop a model for evaluating the characteristics
of composite web services and implement a simulation tool that allows analysis
of their behavior under different conditions.

---

## Features

- Modeling of composite web services
- Simulation of service interaction workflows
- Calculation of service composition metrics
- Analysis of system performance and reliability

---

## Project Structure
project/

- core/           # core logic of the system
-  data/           # input data for simulations
-  output/         # generated simulation output
-   esults/        # processed results and analysis
-   ui/             # user interface modules
-   main.py         # main entry point of the application
-   README.md
